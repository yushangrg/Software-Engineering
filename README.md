# Game Tips and Tricks

## Project Overview
Game Tips and Tricks is a community-driven web application that allows users to share, discover, and discuss tips and strategies for video games. The project focuses on collaboration and knowledge sharing between gamers rather than financial exchange, aligning with the module theme of **sharing, exchange, and building community**.

This project is developed as part of the **Software Engineering** module using an iterative Scrum-based approach.

---

## Project Theme
**Sharing, exchange, and building community**

The platform enables gamers of different skill levels to help each other by contributing gameplay tips and advice in a structured and accessible way.

---

## Technology Stack
- **Backend:** Node.js, Express.js  
- **Database:** MySQL  
- **Frontend:** HTML (served as static files)  
- **DevOps:** Docker, Docker Compose  
- **Version Control:** Git, GitHub  

---

## Project Structure
The project uses a provided scaffolding structure supplied by the module, which has been adapted to reflect the chosen project idea.

Key folders:
- `app/` – Express application logic
- `app/services/` – Database connection
- `static/` – Static frontend files
- `sd2-db.sql` – Database initialisation script
